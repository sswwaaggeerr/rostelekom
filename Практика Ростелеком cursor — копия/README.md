# RTK Brigade Planner (FastAPI + HTML/JS)

MVP для загрузки Excel, проверки данных и планирования задач бригад с учетом:
- обычные бригады: 08:00-17:00, обед 12:00-13:00;
- дежурная бригада: 08:00-20:00, обед 14:00-15:00;
- целевой процент выполнения задач: не менее 82%;
- разрешенный овертайм для части бригад (настраивается в минутах).

## Запуск

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Открыть: <http://127.0.0.1:8000>

## Обязательные колонки Excel

- `task_id`
- `address`
- `visit_time`
- `sla_deadline`
- `task_type`
- `client_type`
- `duration_min`

## Опциональные колонки Excel (для маршрутов)

- `lat` — широта (число)
- `lon` — долгота (число)

Если `lat/lon` заполнены хотя бы у двух задач бригады, в результате появится ссылка на маршрут в Яндекс Картах.
Чтобы считать время/дистанцию с учётом пробок, задайте переменную окружения `YANDEX_ROUTING_API_KEY`
(Routing API v2: `https://api.routing.yandex.net/v2/route`).
